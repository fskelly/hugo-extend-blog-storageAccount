Some more trick and things I have learnt with Service Manager 2010 SP1

March 31, 2011

When creating a new Self Service portal, ensure that you log onto on the server using your “Service Manager” Account, the account configured when first creating the Service Manager Server. The portal will install and seem to work. However, when you open the Portal, you will get HTTP 500. When configuring your Data Warehouse, ensure that the Database name has no “–“ in the name as this is NOT supported by the Reporting Component of Service Manager 2010.
